HTML

  • HTML is the basic structure of a web app
  • An element is a type of content that builds a webpage
  • Types of elements used are Text elements, Image elements, Header elements, List elements, Meta related elements, Content Division elements, and Semantic elements.
  • The head element: contains metadata that helps render the page correctly, it is not visible to the browser.
  • The body element: contains the content shown to the user, all elements to be seen / interacted with belong here
  • The body: is a semantic HTML element (name of the element describes its content & role it plays on the webpage)
  • div element: creates a division
  • br element: creates a break (adds a blank line to break up text, images, or interactables.
  • a element: creates an anchor that creates links to other webpages
  • href element: points to the url of the link
  • img element: allows you to embed pictures onto your webpage
  • src: sources the url for the image embeded on your webpage
  • ul element: creates an unordered list
  • ol element: creates an ordered list
  • li element: creates list items for ul and ol

CSS

  • CSS makes the base structure of html more visually appealing
  • 3 ways of styling a webpage are Inline CSS, Internal CSS style sheet, and External CSS style sheet (most popularly used becuase it allows developers to keep all their CSS rules in a seperate file)
  • style is an attribute/element that effects html elements
  • Inline CSS is used inside of an html element such as h1, p, img etc.
  • Internal CSS is used in the head element of html
  • external CSS style sheet is a seperate file linked to the html file through a link element in the htmls head element
  • class/IDs create a class of element to be effected by CSS
  • selector: element/attribute to which the rules will apply
  • Decleration Block: broken down to 2 important parts the CSS property aswell as the value
  • Margin: indicates the room around the element
  • Padding creates room inside of element effectively making the element larger
  • every html element is a box containing margins on the outside and borders,padding and content on the inside
  • Can combine two selectors into one css with a comma this helps write DRY code (do not repeat yourself)
  • text-align: aligns text to a designated part of the screen according to the value
  • Rel: specifiying relationship between current and lnked element
  • link: creates a connection to a external file
  • href: specifies location (url) of external resource, it is using a reletive path whcih means the location is derived from the current URL(location)
  • a period before a class in style will allow all elements with the same class to be effected at once with CSS
  • width: affects the width of element
  • border: affects border of element
  • box-shadow: applies shadow to element (values of px offset shadow to left/right or top/bottom of element)

Git

  • git status: checks what branch you are currently on
  • git check -b branch name: creates a new branch for you to work on and switches it
  • git clone: creates a copy of your repo to your machine in the current directory you are in
  • git init: creates a new local repo
  • git add filename with angle brackets: creates new file with given file name
  • git commit -m "commit message": commit changes to head (but not yet to the remote rope)
  • git commit -a: commit any files you've added with git add and also commit any files you've changed since then
  • git checkout: switched from one branch to another
  • git branch: lists all branches in your repo
  • git branch -d: deletes the feature branch
  • feature branch: is a branch where you can work on files without effecting other people
  • git add: stages your changes to be commited
